Make sure that each empty square is a zero before running the progam to avoid errors

How to run (in Terminal):
Linux:
        $ gcc sudoku.c -o sudoko
        $ ./sudoku
Windows (Using MinGW or GCC):
        > gcc sudoku.c -o sudoku.exe
        > .\sudoku.exe
MacOS X (Clang):
        % clang sudoku.c -o sudoku
        % ./sudoku
